# 정의
### 1. Hash function Algorithm 

임의의 길이의 입력 데이터를 정해진 길이의 데이터로 매핑하는 함수. 

결과값으로는 기존 입력 데이터를 알 수 없다. 

매핑 과정을 Hash라고 부르며 이때 입력 데이터는 Key, 결과값은 Value가 된다.


### 2. HashMap 

Hash + Map

Key-Value쌍으로 이루어진 연관 배열 (associative array)

# 3. HashMap의 특징
1) null key, null value를 허용
2) 데이터의 순서를        보장하지 않는다
3) 하나의 key는 하나의 value만 가질 수 있다.
4) 중복된 key는 불가하나 중복된 value는 가능하다.
